# Prioritize — your drop-zone (read me)

Put anything here that the coding agent must take into account for THIS project.
The agent checks this folder **at least once per mission** and treats whatever it
finds as **high-priority** guidance that overrides defaults.

## What to drop here

- **Notes & instructions** — any `.md`, `.txt`, `.docx` (or similar) with special
  instructions, design tips, brand rules, copy, research, or "must / must-not" notes.
- **Extra images** — put images that were NOT on the original site into
  `priority-images/` (logos, headshots, product photos, hero imagery, icons…).

## How to help the agent

Name files for what they are and where they go, e.g.
`hero-image-homepage.png`, `brand-voice-notes.md`, `new-logo.svg`,
`pricing-copy-final.docx`. Or just say so inside a notes file.

## If empty

If this folder (and `priority-images/`) is empty when the build starts, the agent
ignores it and proceeds normally. You can add files at any time; the agent re-checks
each mission.
